Kotlin # Introducción # Kotlin es un lenguaje de programación creado para correr sobre la JVM, por lo que el código debe ser compilado a Java bytecode antes de ser ejecutado. Cuenta con otras características como ser transpido a código javascript o compidado a código nativo de otras plataformas como iOS o Windows.
